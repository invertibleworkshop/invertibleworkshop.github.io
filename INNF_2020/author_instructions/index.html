Camera-Ready Instructions

To submit the final camera-ready version of your workshop paper, follow these instructions:
  • Please use this style file. The style file replaces ICML's "appearing in” string with “Second workshop on \textit{Invertible Neural Networks, Normalizing Flows, and Explicit Likelihood Models} (ICML 2020), Virtual Conference”
  • In the main tex file replace "\usepackage{icml2020}” with "\usepackage[accepted]{icml2020}” to submit the camera ready version.
  • Use a single pdf that contains both the main paper and any supplementary material.
